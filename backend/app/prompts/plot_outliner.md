# 情节大纲提示词

## 任务

根据提供的小说原文/摘要和人物档案，生成竖屏短剧改编的分集大纲。

---

## 输出要求

### 基本信息
- 剧集总集数建议（根据原著体量，通常 60–100 集）
- 每集一行大纲

### 大纲格式

```
第X集：[集名]
- 核心事件：发生了什么（一句话）
- 开场钩子：0-10秒如何抓住观众
- 主要冲突：本集最激烈的对抗
- 结尾悬念：用什么钩子引发下一集
- 人物变化：主角或对立面发生了什么变化
```

---

## 结构要求

### 三幕结构（参考黄金比例）

**第一幕（前 15%）：极致开局**
- 第 1–3 集：最强钩子，主角处于最低点（被羞辱/被背叛/最困境）
- 第 3 集：**强反转**——主角身份/能力初显（观众看到希望）

**第二幕（15%–85%）：升级冲突**
- 每隔 5–8 集：一次大的反转或升级
- 第 10 集：**强反转**——敌对关系重大转变或主角重大胜利
- 第 20 集：**强反转**——全剧矛盾激化到最高点

**第三幕（后 15%）：终极决战与收官**
- 大反派最后反扑
- 主角完成最终逆袭
- 情感线收束

### 节奏公式
```
委屈 → 委屈加倍 → 反击信号 → 小胜 → 再次压制 → 大反转 → 全面逆袭
```

---

## 改编取舍原则

### 人物与原著（硬性）

- **姓名、设定、经历、外形、人物关系**必须与原著一致，不得因集数或节奏而改写、替换或臆造。
- 若需删减戏份，只删**出场或情节量**，不改变**人物是谁、做过什么（原著已写明的）**。
- 「人物变化」字段仅描述原著中该阶段**已有**的发展，不可编造原文无据的逆袭/黑化/身世。

### 情节与拍摄（可做）

- 可对情节做取舍、压缩、场景化、调整叙事顺序，以适合竖屏节奏与拍摄；不得编造与原著矛盾的核心事件。

### 必须保留
- 主线矛盾（爱情/商战/复仇等核心冲突）
- 标志性场景（打脸场面、身份揭露场面、复仇场面）
- 核心人物关系线

### 可以删改（仅针对情节与笔墨）

- 支线故事（若不影响主线及原著人物关系可删）
- 大段心理描写（转化为**符合人设**的动作/对白）
- 过渡性情节（可在场景说明中一笔带过）

**不可**：为制造强反转而给角色增加原著未有的身份、关系或经历（除非用户提供书面特许说明）。

### 竖屏特有要求
- 每集必须是独立的情绪单元：有头有尾，不留「过渡集」
- 前 5 集的大纲要最详细（决定观众是否继续看）
- 每集结尾要让观众产生「然后呢」的迫切感

## JSON 输出格式

```json
{
  "adaptation_meta": {
    "original_title": "原著书名",
    "genre": "题材类型（如：女频·都市逆袭复仇）",
    "total_episodes": 80,
    "logline": "一句话概括全剧（≤30字）",
    "hook_premise": "什么让观众从第一集就无法停下来"
  },
  "episodes": [
    {
      "ep": 1,
      "title": "集名",
      "core_event": "核心事件",
      "opening_hook": "开场钩子",
      "main_conflict": "主要冲突",
      "ending_hook": "结尾悬念",
      "character_change": "人物变化",
      "special_note": "特殊要求（如：必须强反转）"
    }
  ]
}
```

## 质量自检清单

在输出大纲前，请自查：
- [ ] 第 1 集的前 10 秒是否有足够强的钩子？
- [ ] 第 3、10、20 集是否安排了强反转？
- [ ] 每集结尾是否都有引发下集的悬念？
- [ ] 主角的逆袭是否有足够的铺垫和层次？
- [ ] 对立面（反派）是否足够嚣张，让观众期待打脸？
