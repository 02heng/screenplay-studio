# 人物分析提示词

## 任务

分析提供的小说原文或摘要，提取并整理所有主要人物的信息，为竖屏短剧改编建立人物档案。**人设、经历、关系、外形均以原著为唯一依据**；仅可在「改编注意事项」中讨论镜头与对白呈现方式，不得建议改写角色本质以迎合套路。

---

## 原著忠实（优先于下文的「短剧特征」描述）

- 所有字段须能在原文中找到依据；不确定则写「原文未详」。
- 不得以「短剧讨喜」为由合并、删减人物或编造身世/外貌/性格。
- 下文的「竖屏短剧人物特征」仅作**在忠于原著前提下**的节奏参考，若与原文冲突则以原文为准。

---

## 输出格式

请以 JSON 格式输出人物档案列表，结构如下：

```json
{
  "characters": [
    {
      "name": "人物姓名",
      "role": "主角/配角/反派/工具人",
      "gender": "男/女",
      "identity": "职业/身份/背景（一句话）",
      "core_motivation": "核心动机（他/她最想要什么）",
      "core_fear": "核心恐惧（他/她最害怕什么）",
      "personality": "性格特点（3个关键词）",
      "speech_habit": "说话习惯（口癖、语气特点）",
      "key_relationships": ["与XXX的关系：描述", "..."],
      "arc": "人物弧线（从...到...的变化）",
      "adaptation_notes": "改编注意事项（与竖屏短剧风格的适配建议）"
    }
  ],
  "relationship_map": "人物关系总结（100字内）",
  "main_conflict": "核心矛盾（主角与什么力量对抗）"
}
```

---

## 分析重点

### 竖屏短剧人物特征
- **主角**：必须有强烈的逆袭/复仇/成长动机；开场即面对极致委屈或困境
- **对立面**：嚣张、不可理喻、给主角制造障碍，后期必须被打脸
- **助力方**：在关键时刻提供信息差或资源，推动主角逆袭

### 对白风格分析
分析每个主要人物的语言特征：
- 惯用短句还是长句
- 是否有口头禅
- 情绪化程度
- 是否话少（留白多）还是话多（密集输出）

### 改编适配建议（仅限拍摄层面）

- 哪些人物的内心戏须转化为**不违背人设**的可拍摄动作或对白节奏
- 哪些关系线在有限集数内需**按原著顺序**保留关键节点（不得改关系性质）
- **禁止**在此建议合并角色、改写人设、编造原文未有的经历或外形

## 注意事项

1. 严格基于提供的原文内容，不要补充原文没有的人物设定；**禁止**为凑套路而改写人物
2. 若原文信息不足，在对应字段注明「原文未详」
3. 人物分析要服务于后续剧本改编，聚焦于**可拍摄的外部冲突**而非编造内心世界
4. 外貌、身份、经历与原著冲突时，**一律以原著为准**
