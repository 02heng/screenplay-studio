# 人物分析提示词

## 任务

分析提供的小说原文或摘要，提取并整理所有主要人物的信息，为竖屏短剧改编建立人物档案。

---

## 输出格式

请以 JSON 格式输出人物档案列表，结构如下：

```json
{
  "characters": [
    {
      "name": "人物姓名",
      "role": "主角/配角/反派/工具人",
      "gender": "男/女",
      "identity": "职业/身份/背景（一句话）",
      "core_motivation": "核心动机（他/她最想要什么）",
      "core_fear": "核心恐惧（他/她最害怕什么）",
      "personality": "性格特点（3个关键词）",
      "speech_habit": "说话习惯（口癖、语气特点）",
      "key_relationships": ["与XXX的关系：描述", "..."],
      "arc": "人物弧线（从...到...的变化）",
      "adaptation_notes": "改编注意事项（与竖屏短剧风格的适配建议）"
    }
  ],
  "relationship_map": "人物关系总结（100字内）",
  "main_conflict": "核心矛盾（主角与什么力量对抗）"
}
```

---

## 分析重点

### 竖屏短剧人物特征
- **主角**：必须有强烈的逆袭/复仇/成长动机；开场即面对极致委屈或困境
- **对立面**：嚣张、不可理喻、给主角制造障碍，后期必须被打脸
- **助力方**：在关键时刻提供信息差或资源，推动主角逆袭

### 对白风格分析
分析每个主要人物的语言特征：
- 惯用短句还是长句
- 是否有口头禅
- 情绪化程度
- 是否话少（留白多）还是话多（密集输出）

### 改编适配建议
- 哪些人物可以合并（节省篇幅）
- 哪些人物的内心戏需要转化为可拍摄动作
- 哪些关系可以加强冲突密度


---

## 注意事项

1. 严格基于提供的原文内容，不要补充原文没有的人物设定
2. 若原文信息不足，在对应字段注明「原文未详」
3. 人物分析要服务于后续剧本改编，聚焦于**可拍摄的外部冲突**而非内心世界
